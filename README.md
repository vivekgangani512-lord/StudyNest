# StudyNest

StudyNest is a modern student productivity platform designed to help students organize their academic life. It provides an intuitive interface for managing notes, study materials, and learning resources.

## Live Demo

https://studynest-eosin.vercel.app

## GitHub Repository

https://github.com/vivekgangani512-lord/StudyNest

## Features

- Modern Responsive UI
- Clean Dashboard
- Study Resource Management
- Fast Performance
- Mobile Friendly
- Built with React + Vite

## Tech Stack

- React.js
- Vite
- JavaScript
- HTML5
- CSS3
